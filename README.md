# mellea-tutorials

[Mellea](https://github.com/generative-computing/mellea) is a library for writing generative programs — structured, verifiable code that calls LLMs as first-class components rather than wrapping prompts in strings.

This repo is home to long-form demos and tutorials for Mellea. The main Mellea repo keeps its examples short and focused; the notebooks here have more room to walk through end-to-end workflows, conference material, and exploratory demos.
